# 沙盒建造游戏功能文档

## 1. 产品概述

本项目是一款第一人称方块沙盒游戏。玩家从零物资开始，在可破坏的自然环境中采集资源、建造建筑、合成物品，并应对饥饿、溺水、坠落和夜间怪物等生存挑战。

## 2. 已实现功能

### 2.1 玩家操作

- `WASD` 移动，鼠标控制视角，空格跳跃。
- `Shift` 潜行并防止玩家从方块边缘跌落。
- 数字键 `1–9` 切换快捷栏，`E` 打开或关闭27格背包。
- 右键放置方块或使用门、床和熔炉；左键攻击生物，长按左键挖掘方块。
- 玩家可以在水中游泳，长按空格上浮；水中移动速度降低。

### 2.2 方块与物品

当前共有18种物品：石头、木头、玻璃、木门、脚手架、草地、水、红砖、灯块、树叶、原木、泥土、沙子、熔炉、肉、床、煤炭和火把。

- 方块按照网格放置，不能放进玩家身体或已占用的位置。
- 方块具有不同挖掘时间，挖掘过程显示渐进裂纹。
- 方块破坏后生成掉落物，玩家靠近后自动拾取。
- 木门可开关，脚手架可攀爬，床占用两个方格。
- 火把与灯块会产生光照；照明范围内不会生成怪物。
- 背包只显示数量大于零的物品，快捷栏显示物品图标和数量。

### 2.3 世界与地形

- 地面随玩家移动持续扩展，采用分帧生成以降低卡顿。
- 地层由两层草地方块、两层石头和黑色不可破坏底层组成。
- 世界包含草丘、石堆、树木、灌木、林地、岩石区和灌木区。
- 池塘为不规则形状，水面齐平，内部包含1～3格深度。
- 池塘岸边生成沙子，底部及侧壁使用泥土、石头或沙子封闭。
- 天空包含移动云彩、太阳和月亮；完整昼夜循环为10分钟。

### 2.4 水系统

- 水具有流动纹理，并会向低处和相邻空位扩散。
- 多股水流汇合时水量叠加，但高度最高为一格。
- 只有满一格的水可以汲取。
- 移除水源时，关联水流先降低水位再消失。
- 排水后，相邻满水可以重新填充空位。
- 水流可以推动玩家和生物，玩家也可以挤动生物。

### 2.5 生物与战斗

友好生物包括牛、羊、猪、鸡和兔子。所有生物拥有10点生命值、随机行走行为、碰撞体及步行动画。

- 玩家空手攻击力为1点，攻击会产生击退。
- 友好生物死亡后按种类掉落不同数量的通用肉。
- 生物每分钟在玩家视线外生成，总数量上限为40只。
- 怪物只在夜间生成，会追踪并攻击18格内的玩家。
- 白天现存怪物会消失；灯和火把周围8格内不会生成怪物。

### 2.6 生存系统

- 玩家拥有20点生命值和20点饥饿值。
- 肉可通过右键食用，每块恢复2点饥饿值。
- 饥饿值归零后，每3秒损失1点生命。
- 玩家水下氧气上限为15秒，耗尽后每秒受到伤害。
- 从超过3格的高度落地会受到坠落伤害，落水可中止坠落计算。
- 生命耗尽后返回出生点，生命恢复至20点，饥饿恢复至10点。
- HUD显示生命、饥饿、游戏时间和必要时的氧气值。

### 2.7 合成与熔炉

| 输入 | 输出 |
| --- | --- |
| 1原木 | 4木头 |
| 6木头 | 3木门 |
| 4木头 | 4脚手架 |
| 9石头 | 1熔炉 |
| 6木头 | 1床 |
| 1煤炭 + 1木头 | 4火把 |
| 1火把 + 1玻璃 + 1石头 | 1灯块 |

熔炉每次烧制耗时3秒，支持以下配方：

- 1沙子 → 1玻璃。
- 1木头 → 1煤炭。

### 2.8 睡眠

- 玩家夜晚右键床可以睡觉。
- 睡眠后时间直接跳到次日早晨。
- 白天使用床不会改变时间。

## 3. 部分实现或暂未启用

- 动态自然景观及动态水域的生成函数已经存在，但没有接入定时触发。
- 中央事件提示接口仍然保留，但当前不显示黄色提示文字。
- 地面可持续扩展，但树木、矿石和池塘不会随玩家探索持续生成。
- 熔炉采用全局烧制状态，多个熔炉尚未拥有独立进度和库存。

## 4. 后续候选功能

- 世界、玩家位置、背包和建筑的存档与读档。
- 工具、武器、护甲、耐久度及不同挖掘效率。
- 牛肉、羊肉、猪肉和鸡肉等独立食物，以及熟食烧制。
- 工作台、更完整的配方系统和熔炉燃料机制。
- 更多怪物、生物掉落、繁殖和生态数量控制。
- 玩家死亡掉落、经验、状态效果和重生点设置。
- 音效、背景音乐、暂停菜单、按键设置及画面设置。
- 顾客订单、房屋验收、交易和商店经济系统。
- 多人游戏、地图种子及程序化资源持续生成。

## 5. 当前版本限制

- 开局背包为空，玩家必须从自然环境获取第一批资源。
- 肉目前为统一物品，没有区分来源，也不能烹饪。
- 怪物只有一种，且不会掉落物品。
- 没有自动存档，退出游戏后世界状态不会保留。
- 项目尚未建立自动化测试和正式导出流程。
